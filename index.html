Created First Repository
July 15th, 2025

Fiona's first GitHub project, my-first-project, is a Coding Club website for kids aged 8 to 18. The site features navigation, an about section, sign-up form, contact section, and footer. It helps parents and children discover fun coding activities and resources. Fiona credits her mentor Rudy Matheson for support and used creative social media icon animations in the project.

Created Bootstrap Project: Mei Wong
July 24th, 2025

Fiona launched bootstrap-project-mei-wong exactly three weeks ago. This project showcases her skills in HTML and CSS, using Bootstrap to build a modern, responsive website. The repository includes a well-structured layout and demonstrates Fiona's growing expertise in front-end development.

Created Boardwalk Games Project
July 28th, 2025

Fiona launched boardwalk-games just a few days after her Mei Wong project. This site features a booking system, game library, and success page, all built with HTML and CSS. The project highlights Fiona's ability to design interactive, user-friendly web applications and her rapid progress in front-end development.

Created Love Running Project
August 1st, 2025

Fiona created love-running a few days after Boardwalk Games. This project is a website for running enthusiasts, featuring a gallery, sign-up page, and resources for the running community. Built with HTML and CSS, it highlights Fiona's ability to design engaging, user-focused web experiences.

Created Major Project: Safeguarding Matters
August 5th, 2025

Fiona launched Safeguarding Matters, her first major project, just days after Love Running. This website provides clear, accessible information about safeguarding practices, signs to watch for, and action steps for concerned individuals. Built with HTML, CSS, and Bootstrap, it features a calm, professional design and includes user stories, wireframes, and thorough testing. Fiona used AI tools for images, content, and debugging, and reflects on the importance of planning and workflow in her project write-up.
